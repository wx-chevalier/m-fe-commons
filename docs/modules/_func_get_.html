RequiredRecursively

RequiredRecursively<T>: Exclude<T extends string | number | boolean ? T : {}, null | undefined>
  • Defined in Desktop/Workspace/Github/m-fe/m-fe-commons/packages/utils/src/func/get.ts:34

Removes all undefined and nulls from the nested types making all nested fields non-nullable hence you can simply chain through it and autosuggestion will work and will not complain on undefiness and nullability of some of the nested fields including the root

example

type AllOpt = { a?: string arr?: string[] | null b?: { c?: string | undefined | null n?: | { a?: string | null b: object | null } | null | undefined } c?: string }

type AllReq = RequireRecursively<AllOpt | undefined | null>

const obj: AllReq = ({} as unknown) as AllReq

obj.b.c // -> string obj.b.n.a // -> string obj.b.n.b // -> object obj.arr // -> string[]
